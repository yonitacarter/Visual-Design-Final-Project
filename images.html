Images

I have manipulated and optimised six images and have saved them for the web, as per requirements.

Image 1

Logo for web

AI file

Software: Adobe Illustrator

Font: Bauhaus 93

Colours: logo dot #B10166, logo letters #290151

I chose the letters Y B E (first letters of my website title) and created an outline (grouped image) which I then ungrouped, in order to be able to manipulate each letter individually. I then positioned the Y above the B and aligned the sides and the size by using the direct selection tool. I used the shape builder tool to clip the end of the letter E. The final image comprising of the combination of letters and the different colour dot is also grouped.

Image saved for web in png format, dimensions 2591x1574, size 56.3KB

Image 2

Original image

Original image - downloaded from Unspash.com

Software: Adobe Photoshop

Filter – Camera Raw 13.0

Temperature – reduced to -26 to achieve the blue hue.

Duplicated the picture and then edited in Quick Mask mode to select the backpack for colour manipulation. I achieved this by putting on a mask, created new Fill or Adjustment layer and chose colour balance and changed to match the logo dot colour. The colour balance layer is given a clipping mask to affect only the backpack and not the entire image.

Image is saved for web in jpg format, dimensions 1950x1300, size 339KB
